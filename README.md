# Bill_of_Sale_Firearm_North_Carolina_WebApp
The full functioning Web Application Bill of Sale for a Firearm in North Carolina

This application yields a Bill of Sale for a Firearm in North Carolina.  The users (Seller/Buyer) has the option of using the Bill of Sale in one of four modes: 

Mode 1> a blank PDF form to be completed entirely manually by the Seller/Buyer
Mode 2> same as Mode 1, plus a web page with tooltips, checkboxes, radio buttons, dropdown list that allow the Seller/Buyer to embed transaction data and affirmations.  It also allows Seller and Buyer to print this collection of data with two blank suitable Seller/Buyer signature of affirmation blocks.
Mode 3> same as Mode 2 plus Mode 3 allows the Seller/Buyer signatures to be captured electronically, filed, and printer along with the previously collected data.
Mode4> same as Mode 3, plus it gives the Seller/Buyer opportunity to email to the selected addressees the entire raw data collected, and electonic rendering of the two signature blocks, presented drivers licenses, conceal cary handgun permits, a PDF of the entire Mode 3 version, and ancillary documents supporting the Firearm transaction.
