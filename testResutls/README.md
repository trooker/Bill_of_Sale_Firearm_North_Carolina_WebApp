# Contents of testResults Folder

This folder contains the contents that a Seller/Buyer could expect when using the Bill_of_Sale_Firearm_NC_991-995.html coupled with the inputs of the Seller's NC Driver's License, Seller's Conceal Carry Handgun permit, and Buyer's NC Driver's License.  The two artifacts in the "testResults" folder are PDF files:

TestResults 1.: 991_995_Bill-Of-Sale_Feedback_Form.pdf.  This document shows the Seller/Buyer responses to the web application questions and responses for affirmation,  It also presents electronic rendering of the actual signatures as captured by the web application. 

TestResults 2: 991_995_Bill_of_Sale_for_Firearm _North_Carolina.pdf. This document displays a human readable bill of sale for a Firearm in North Carolina.  It is soley suited only for the current stautes of North Carolina as they exist today 1/02/2025.
